## ⚖️ 表三：AI 互博提示词权限与约束权重 (Prompts Weighting)
**目的**：控制对话走向，确保 AI 专家通过“剥洋葱”式提问提取深层知识。

| 逻辑维度 | AI 专家 (Expert Agent) 约束 | AI 小白 (Novice User) 约束 |
| :--- | :--- | :--- |
| **开场权限** | 禁止自我介绍；必须先通过 B/C 维度进行事实确认。 | 必须带入 E 维度的情绪。第一句内容禁止涉及 A 以外的事实。 |
| **追问因子** | **强制性诊断**：至少完成 2 次追问。必须体现“排除法”逻辑。 | **非对称性隐藏**：只有在对方给出“对应价值”时才透露具体数据。 |
| **术语禁令** | **绝对剥夺**：前 3 轮禁止使用“服务、签约、报价、流程”等采购词汇。 | **防火墙指令**：严禁使用任何 REC 表格中的术语。违体即强制刷新。 |
| **结案分数** | 置信度 > 0.9 且匹配 REC 分类，方可给出 concluded 信号。 | 只有专家解决了 C 和 E 的连带关系，才能表达入账意向。 |
